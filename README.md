## Run Locally

Clone the project

```bash
  git clone https://github.com/4mubarak/Smart-shop.git
```

Go to the project directory

```bash
  cd Smart-shop
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
